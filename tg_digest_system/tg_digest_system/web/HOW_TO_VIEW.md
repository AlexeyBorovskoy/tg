# Как посмотреть веб-формы

## Вариант 1: Запуск локально (рекомендуется)

### Шаг 1: Установить зависимости
```bash
cd tg_digest_system/tg_digest_system/web
pip install -r requirements.txt
```

### Шаг 2: Настроить переменные окружения
Убедитесь что есть файл `.env` в корне проекта с настройками БД и Telegram:
```bash
# Минимум для просмотра форм (БД должна быть доступна)
PGHOST=localhost
PGPORT=5432
PGDATABASE=tg_digest
PGUSER=tg_digest
PGPASSWORD=your_password

# Telegram (для проверки доступа к чатам)
TG_API_ID=your_api_id
TG_API_HASH=your_api_hash
TG_SESSION_FILE=/path/to/session
TG_BOT_TOKEN=your_bot_token
```

### Шаг 3: Запустить веб-сервер
```bash
cd tg_digest_system/tg_digest_system/web
python web_api.py
```

Или через uvicorn:
```bash
uvicorn web_api:app --host 0.0.0.0 --port 8080 --reload
```

### Шаг 4: Открыть в браузере
- Главная страница: http://localhost:8080/
- Список каналов: http://localhost:8080/channels?user_telegram_id=499412926

## Вариант 2: Сохранить страницу для просмотра

### Способ 1: Сохранить HTML через браузер
1. Откройте страницу в браузере
2. Нажмите `Ctrl+S` (или `Cmd+S` на Mac)
3. Выберите "Веб-страница, полностью" (Web Page, Complete)
4. Сохраните файл (например, `index.html` и `channels.html`)
5. Отправьте мне эти файлы

### Способ 2: Скриншоты
1. Сделайте скриншоты форм
2. Сохраните изображения
3. Отправьте мне файлы изображений

### Способ 3: Инспектор браузера
1. Откройте страницу в браузере
2. Нажмите `F12` (открыть DevTools)
3. Вкладка "Elements" → правый клик на `<html>` → "Copy" → "Copy outerHTML"
4. Вставьте HTML в файл и отправьте мне

## Что мне нужно для стилизации

Для создания корпоративного стиля мне нужно:

1. **HTML файлы** (если хотите показать текущее состояние):
   - `templates/index.html`
   - `templates/channels.html`

2. **Скриншот референса** (уже есть):
   - Вы можете просто описать что именно вам нравится в корпоративном стиле
   - Или указать конкретные элементы которые нужно применить

3. **Требования**:
   - Какие цвета использовать (из вашего корпоративного стиля)
   - Какие шрифты
   - Какие иконки (если есть библиотека)
   - Структура (сайдбар, шапка, основной контент)

## Текущее состояние форм

Сейчас формы имеют:
- Фиолетовый градиентный фон
- Белые карточки с формами
- Современный дизайн с тенями и анимациями
- Простая структура без сайдбара

Для корпоративного стиля нужно:
- Светлый фон (белый/светло-серый)
- Темный текст
- Синие акценты для активных элементов
- Более строгая структура
- Возможно сайдбар для навигации
- Таблицы вместо карточек для списков
