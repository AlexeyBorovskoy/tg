Инструкция авторизации Telethon

  1. Зайдите на my.telegram.org, создайте приложение, получите `api_id/api_hash`.
  2. Сохраните данные в форме слева.
  3. Выполните команду ниже в терминале.
  4. Введите номер телефона и код из Telegram.
  5. Проверьте, что `.session` создан.
cd tg_digest_system/scripts TG_API_ID=12345678 \ TG_API_HASH=0123456789abcdef0123456789abcdef \ TG_SESSION_FILE=/app/data/user-sessions/user_7.session \ python3 auth_telethon.py
Пока нет session-файла, сервис не сможет читать пользовательские чаты.
