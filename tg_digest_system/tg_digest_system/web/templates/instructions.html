Как получить необходимые данные

Пошаговая инструкция для новых пользователей

Важно: Система работает только с чатами, где она присутствует. Если вы хотите добавить чат, убедитесь что система имеет к нему доступ.
1
Узнать свой Telegram ID
Вам понадобится ваш Telegram ID для идентификации в системе. Есть несколько способов:
Способ 1: Через бота @userinfobot
  1. Откройте Telegram и найдите бота @userinfobot
  2. Нажмите кнопку "Start" или отправьте команду /start
  3. Бот пришлёт вам ваш Telegram ID (число, например: 499412926)
Способ 2: Через бота @RawDataBot
  1. Откройте Telegram и найдите бота @RawDataBot
  2. Нажмите кнопку "Start"
  3. Бот пришлёт JSON с данными вашего аккаунта
  4. Найдите поле "id" - это ваш Telegram ID
Пример: Ваш Telegram ID может выглядеть так: 499412926
2
Узнать ID чата или канала
Для добавления чата в систему вам нужен его Telegram ID. Способ зависит от типа чата:
Для публичных каналов:
  1. Откройте канал в Telegram
  2. Скопируйте ссылку на канал (например: https://t.me/channel_name)
  3. Или используйте веб-версию Telegram: https://web.telegram.org
  4. В URL будет виден ID канала (например: #-1001234567890)
Для приватных групп и каналов:
  1. Откройте группу/канал в Telegram
  2. Перейдите в настройки группы/канала
  3. Найдите раздел "Информация" или "About"
  4. Или используйте бота @RawDataBot:
    • Перешлите любое сообщение из группы/канала боту
    • Бот пришлёт данные сообщения
    • Найдите поле "chat" → "id" - это ID чата
Важно: ID каналов обычно начинается с -100 (например: -1001234567890). ID групп может быть положительным числом (например: 5228538198).
Примеры ID: • Канал: -1001234567890 • Группа: 5228538198
3
Указать куда присылать дайджесты
Вы можете получать дайджесты в личные сообщения или через бота:
Вариант 1: Личные сообщения
  1. Используйте ваш Telegram ID (из шага 1)
  2. Или ID другого пользователя, которому нужно отправлять дайджесты
Вариант 2: Через бота
  1. Создайте бота через @BotFather
  2. Получите токен бота
  3. Узнайте ID бота через @userinfobot
  4. Используйте ID бота как получателя
Примеры получателей: • Ваш личный ID: 499412926 • ID бота: 8572555788
4
Проверить доступ системы к чату
Система работает только с чатами, где она присутствует. Перед добавлением убедитесь:
  1. Система должна быть участником чата/канала
  2. Для приватных групп система должна быть добавлена администратором
  3. Если система не имеет доступа, вы получите сообщение об ошибке при добавлении
Ограничение тестовой версии: В настоящее время система находится в стадии тестирования и поддерживает работу только с чатами, где она присутствует. Работа со сторонними ресурсами пока не поддерживается.
5
Добавить канал в систему
После получения всех данных:
  1. Перейдите на страницу добавления канала
  2. Заполните форму:
    • Ваш Telegram ID
    • Telegram ID чата для мониторинга
    • Название чата (опционально)
    • Куда отправлять дайджесты (Telegram ID получателя)
    • Имя получателя (опционально)
  3. Нажмите "Добавить канал"
  4. Система проверит доступ и начнёт обработку
После добавления система автоматически: • Загрузит всю историю чата • Обработает изображения через OCR • Создаст сводный инженерный документ • Начнёт отправлять дайджесты указанному получателю
info Дополнительная информация
Скачивание инженерного документа:
  • После обработки чата система создаст сводный инженерный документ
  • Вы можете скачать его на странице "Мои каналы"
  • Нажмите кнопку "Документ" рядом с нужным каналом
Просмотр дайджестов:
  • Все дайджесты отправляются указанному получателю
  • Вы можете просмотреть историю дайджестов на странице каналов
