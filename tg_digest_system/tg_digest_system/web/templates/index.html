info Как это работает

  • Система автоматически загрузит всю историю чата
  • Обработает все изображения через OCR
  • Создаст сводный инженерный документ
  • Начнёт отправлять дайджесты в указанный чат или боту

info Ограничения тестовой версии

В настоящее время система находится в стадии тестирования и поддерживает работу только с чатами, где она присутствует. Работа со сторонними ресурсами пока не поддерживается.
