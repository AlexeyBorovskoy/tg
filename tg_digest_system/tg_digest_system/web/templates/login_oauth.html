Кнопка «Войти через Яндекс» появится после настройки OAuth.

На сервере в каталоге docker создайте или отредактируйте файл secrets.env и задайте переменные:

  • YANDEX_OAUTH_CLIENT_ID, YANDEX_OAUTH_CLIENT_SECRET — из кабинета Яндекс OAuth;
  • BASE_URL=http://93.77.185.71:8000 (адрес приложения, без слэша в конце).

После сохранения выполните: docker compose up -d --force-recreate web. Подробнее — README, раздел «OAuth Яндекс».
