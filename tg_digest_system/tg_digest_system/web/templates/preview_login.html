Как работает проверка доступа:

  1. Вы вводите ваш Telegram ID (например: 499412926)
  2. Система создаёт сессию и сохраняет её в БД
  3. Браузер получает cookie с токеном сессии
  4. При следующих запросах система проверяет cookie
  5. Если cookie валиден → вы авторизованы
  6. Если cookie нет или истёк → нужно войти заново

⚠️ Важно: Система НЕ проверяет может ли пользователь работать. Любой кто знает Telegram ID может войти. Это нормально для внутреннего использования.
